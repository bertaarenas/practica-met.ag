# Juego de Combate – 2 Jugadores

Este proyecto es un videojuego de combate por turnos diseñado para dos jugadores en java. Cada jugador controla un personaje de una de los tres tipos disponibles: Guerrero, Mago o Caballero. Cada clase tiene características distintas de vida, mana y fuerza de ataque, lo que hace que cada elección afecte al combate de manera diferente.

Durante el juego, los jugadores se turnan para realizar acciones, que pueden ser atacar al oponente, usar magia o curarse. Los ataques cuerpo a cuerpo pueden fallar, y las habilidades de magia y curación consumen mana, que se va regenerando en cada turno. El objetivo es reducir la vida del oponente a cero para ganar la partida.

El juego muestra constantemente el estado de cada jugador, incluyendo su vida y mana, para la toma de decisiones.
